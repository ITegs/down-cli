> A CLI that tests if a website is down

<img alt="isdown" src="assets/isdown.png" width="600">

## Install

```bash
$ git clone https://github.com/ITegs/down-cli.git
$ cd down-cli
$ npm install
$ npm link
```

## Usage

```bash
$ isdown <url> [options]

## Options

  -v, --verbose  verbose output
```
